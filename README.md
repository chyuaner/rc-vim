元兒～的 Vim 設定檔
===



## 使用方法
1. 將`gvimrc`複製到你的家目錄，並改名為`.gvimrc`
2. 將`vimrc`複製到你的家目錄，並改名為`.vimrc`
3. 在家目錄開資料夾為`.vim`，將此專案內剩餘檔案全部移到`.vim`資料夾裡面。
4. 打開`vim`測試看看吧 (笑
